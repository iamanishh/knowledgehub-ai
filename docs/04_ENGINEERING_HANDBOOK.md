# Engineering Handbook

## Principles

- Routers coordinate requests.
- Services execute business logic.
- Validators validate.
- FileManager performs file operations.
- Test behavior, not implementation.
- Never trust user-provided filenames.
- Prefer parameterized logging.
- Never log sensitive information.
